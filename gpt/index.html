Halo Sobat! Sebelumnya, terimakasih telah mengunjungi website ini. Saya masih belajar dalam membuat dan mendesign suatu web. Saya buat website ini untuk diri pribadi serta koleksi. Ada beberapa website yang saya buat, seperti tentang pernikahan, bisnis, dan portofolio. Namun saya tidak menyimpan linknya. Jadi hilang entah kemana. Saya cek di github, perlu login. Malas!

Baiklah, sebenarnya apasih itu website? Website adalah kumpulan halaman web atau lokasi di internet tempat Anda menyimpan informasi dan menyajikannya agar bisa diakses oleh siapa pun secara online. Informasi ini bisa tentang diri Anda, bisnis, atau bahkan topik yang Anda minati. Lalu untuk membuat website seperti ini, pentingnya kita belajar bahasa pemrograman. Ada beberapa bahasa pemrograman yang sering digunakan seperti, Python, C#, C++, JavaScript, PHP, Swift, Java, Go, SQL, Ruby. Bahasa pemrograman memungkinkan penggunaan logika seperti “if-else” atau “switch-case” untuk mengambil keputusan berdasarkan kondisi tertentu. Tapi untuk membuat website, Anda harus pelajari HTML dan CSS dahulu.

Kamu bisa membuat website dengan imajinasi mu sendiri. Jika kamu paham bahasa pemrograman, kamu bisa saja membuat website dengan imajinasi mu lalu tuangkan dalam bentuk kode. Tapi bisa aja nih kamu buat website pakai chat gpt atau AI lainnya. Kamu hanya deskripsikan bagaimana website mu. Kalau susah, bisa kamu ambil contoh gambar halaman website lalu berikan kepada chat gpt dan minta agar buatkan website seperti itu. Walau kamu pakai chat gpt, kamu perlu juga belajar. Karena jika kamu tidak mengerti apa fungsi dari kode kode yang diberikan chat gpt, bisa saja website mu jadi jelek.
